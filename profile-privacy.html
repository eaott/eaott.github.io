Privacy Policy

evanott.com does not store any data uploaded through the Custom Profile Filter application (evanott.com/profile-filter.html). Period. This website is hosted as a static page using GitHub Pages. It does not interact with a personal server in any way. The result of the profile filtering is computed entirely on your computer (or phone or tablet), and never stored unless you store it yourself on your own device. This holds whether you let the app sync to your Facebook profile or whether you upload an image manually. If you authorize the Facebook app, evanott.com will have access to the default approved items (email, public profile picture, list of friends who also use the app). This information is not stored. evanott.com does not want the responsiblity of having to secure your data, so it doesn't store it at all.
